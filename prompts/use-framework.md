# Use Baton

This project has Baton installed in `.baton/`.

You are the orchestrator. Read `.baton/orchestrator.md`, then run
`.baton/baton orchestrator brief --phase start`. Show the user the
brief's Harness memory choices and let them choose before planning. Then plan
and use Baton workers for the coding goal; do not implement the goal yourself.
Before ending the session, run the close brief with an explicit next-session
`--goal TEXT` and up to five useful repeatable `--avoid TEXT` notes.
